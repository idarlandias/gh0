## Como Contribuir

Quer contribuir para esta documentação? Veja o [Guia de Contribuição](../CONTRIBUTING.md)!

1. Escolha uma [issue disponível](../../issues)
2. Comente para reivindicar
3. Faça um fork e clone o repositório
4. Adicione seu conteúdo seguindo as instruções no arquivo
5. Abra um Pull Request

## Padrão de Qualidade

Toda contribuição deve:

- ✅ Ser clara e didática
- ✅ Incluir exemplos práticos
- ✅ Estar bem formatada (Markdown correto)
- ✅ Ter ortografia e gramática corretas
- ✅ Citar fontes quando usar informação externa
- ✅ Ser revisada por pelo menos um colega
